# Security Policy

---

# English

Thank you for helping improve project security.

If you discover a security vulnerability, please do not open a public Issue.

Instead, contact the maintainers privately.

We will investigate and respond as soon as possible.

---

# Português

Obrigado por ajudar na segurança do projeto.

Caso encontre uma vulnerabilidade, não abra uma Issue pública.

Entre em contato com os mantenedores de forma privada.

Investigaremos o problema o mais rápido possível.
